# layout/ — Page Shells & Layout Wrappers

Components that provide consistent page structure.

Intended contents:
- `PageContainer.tsx` — enforces the canonical `pt-[52px] py-16 px-6 max-w-7xl mx-auto` (and narrow variant for search/results)
- Any future shared layout primitives (e.g., two-column wiki layout, table of contents sidebar, etc.)

These can compose components from `ui/`.
